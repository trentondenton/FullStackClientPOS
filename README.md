## Installation

1. Install Node.js
2. Install NPM
3. npm install

## Usage

1. npm start
2. Open [http://localhost:3000] in your browser
3. Company Login
   - username: testco
   - password: testing123
4. User Login
   - username: testuser
   - password: testing123

## Build

1. npm run build
2. Copy the build folder to the server
3. Run the server
